## 什么是指纹浏览器？为什么要用它？

先说个基础概念。你每次上网，浏览器其实会"泄露"一大堆信息给网站——操作系统版本、屏幕分辨率、装了哪些插件、用的什么字体，甚至你的 GPU 型号。这些信息组合起来，就形成了独一无二的"浏览器指纹"。

网站通过这些指纹，能轻松识别出"这个人又来了"，就算你换了 IP、清了 Cookie 也没用。对于需要管理多个账号的人来说，这就是个大麻烦——平台一旦发现多个账号来自同一个浏览器指纹，直接给你来个批量封禁。

指纹浏览器的作用，就是给每个账号创建完全独立的浏览环境。每个环境有自己的指纹参数，从网站的角度看，这些账号就像是来自不同的设备和用户，自然也就没法关联了。
